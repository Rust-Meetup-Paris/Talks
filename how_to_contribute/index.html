How to contribute to Rust.

Axel Viala - Rust Meetup Paris - MozSpace

So you want to contribute to Rust?

Rust is developed by a really dynamic and inclusive community, who is very open to newcomers. And the best way to contribute is to enter this community.

How to be a Rustacean?

To Became a Rustacean you need to now how the community is organized:

  • IRC: #rust on irc.mozilla.org
  • Mailling list: https://mail.mozilla.org/listinfo/rust-dev
  • Reddit: http://www.reddit.com/r/rust/.
  • And the GitHub issue tracker.

I want to do something this project is so good!

    First Step building Rust:
  • git clone https://github.com/mozilla/rust && cd rust
    • Un*x way: -> Prerequisites.
    • ./configure [--prefix=Where_You_want_Rust] [--enable-clang]
    • make && make install
    • rustc hello.rs
  • Windows way -> cookbook here.

What can I do now?

    Second step Learning and Hacking Rust :
  • Official Tutorial -> http://static.rust-lang.org/doc/master/tutorial.html
  • Rust for Rubyist(And others...) -> http://www.rustforrubyists.com/
  • Felix's talk at CodeMeshLondon2013
  • Hacking around and reading code -> libcollection::dlist is a good starting points.
  • Writing your own program/crate in Rust.
Tips: Go on IRC and ask for help if you're struggling. ;)

I know Rust, can I contribute ?

    Third step contributing :
  • Pick a bug, and work on it! -> a ton of bugs is waiting for you!
  • Improve doc! As you can see libsync lacks of examples. :(
  • Writing Rust code. -> Plenty of good projects are currently develloped.
  • Writing tests on existing code.
  • Contribute to Servo. (They are really cool!)
  • Spread the word!
    • Tips: There is easy bugs both on Servo and Rust. ;)
